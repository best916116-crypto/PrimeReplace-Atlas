How to read this page

Downstream-CDS values are transcript-boundary hypotheses. High downstream coverage can be boundary-driven and should be read with payload and context burden.

PrimeReplace Atlas is a pre-implementation transcript-architecture resource. Record coverage is not patient coverage.
