1. Search the gene

Use the gene search page after a disease gene or research locus has already been identified.

2. Read the class label

The label summarizes record distribution and payload/context interpretation. It is not a treatment recommendation.

3. Check the unit metrics

Compare single-exon, adjacent exon-block, and downstream-CDS record coverage.

4. Carry forward only as a hypothesis

Separate design, boundary, assay, delivery, and validation work is required before any implementation claim.

PrimeReplace Atlas is a pre-implementation transcript-architecture resource. Record coverage is not patient coverage.
