Transcript units

The browser summarizes three unit types: single exon, adjacent exon block, and downstream-CDS boundary hypotheses.

Record coverage

For a transcript unit, ClinVar record coverage is k/N, where N is the number of gene-level ClinVar P/LP records and k is the number addressed by the unit definition.

Interpretation gates

Low-record-burden<20 ClinVar P/LP records. Genes are shown but interpretation is limited.
Compact payload bin≤5 kb, used as a descriptive payload bin only.
Large-payload bin>20 kb, used as a donor-burden descriptor only.

These gates are interpretation aids, not editing feasibility rules.

PrimeReplace Atlas Browser release 1.0.0. Record coverage is not patient coverage.
