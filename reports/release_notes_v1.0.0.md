# PrimeReplace Atlas release notes v1.0.0

PrimeReplace Atlas is a pre-implementation transcript-unit target-architecture database for disease-associated large-fragment genome writing.

Public browser: https://best916116-crypto.github.io/PrimeReplace-Atlas/  
Repository: https://github.com/best916116-crypto/PrimeReplace-Atlas  
Zenodo: https://doi.org/10.5281/zenodo.20174922

The release includes 5,738 all-mapped genes, 437,301 transcript-unit opportunity rows, 3,459 low-record-burden interpretation-limited genes, 2,279 architecture-interpretable genes, 3,389 high-support condition-associated record groups, 1,278 high-support condition groups with recovered unit coverage, and 160,125 condition-unit coverage rows.
